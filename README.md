# TellUrStori Internship

Onboarding materials and learning resources for TellUrStori interns.

## Overview

This repository contains self-guided learning materials designed to help interns build a strong foundation in web technology fundamentals and Unix terminal workflows. The curriculum is designed for smart beginners with no prior programming experience.

## Getting Started

1. **Read the onboarding document**: Start with [`onboarding.md`](onboarding.md) and follow the roadmaps sequentially.

2. **Complete the tasks**: Look for **‚ñ∂ Try This** sections throughout the document and actually do them.

3. **Validate your work**: Use the [W3C HTML Validator](https://validator.w3.org/) and [CSS Validator](https://jigsaw.w3.org/css-validator/) to check your code.

4. **Reference the example**: Check out [`index.html`](index.html) to see a complete, validated example homepage.

## Learning Approach

- **Self-paced**: Work through materials at your own pace during evenings/weekends
- **Hands-on**: Understanding comes from doing, not just reading
- **Validated**: All HTML examples are validated and follow web standards
- **Practical**: Concepts connect directly to real-world app development
